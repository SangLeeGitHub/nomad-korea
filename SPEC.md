## Phase 1: 핵심 기능 완성 (필터링 & 검색)

- [x] FilterBar에서 상위 컴포넌트로 필터 상태를 전달하는 콜백 구현
- [x] 도시 이름 검색 기능 구현 (한글/영문 검색)
- [x] 월 비용 범위 슬라이더 필터 구현 (최소~최대 만원 단위)
- [x] 정렬 기능 구현 (종합점수, 비용, 카페수, 인터넷속도, 노마드수)
- [x] 필터 초기화 버튼 기능 구현
- [x] CityGrid에 리스트 뷰 옵션 추가 (그리드/리스트 토글)
- [x] 필터 결과가 없을 때 "결과 없음" UI 표시
- [ ] URL 쿼리 파라미터로 필터 상태 유지 (선택사항)
